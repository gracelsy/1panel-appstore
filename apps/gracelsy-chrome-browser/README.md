[Chrome](https://www.google.com/chrome/) 浏览器，也称为 Chrome Browser 或简称为 Chrome，是由 Google Corporation 开发的一款免费开源的网页浏览器。Chrome 使用 Blink 布局引擎来渲染网页，该引擎实现了当前和预期的网页标准。

## 支持的架构


简单地拉取 `linuxserver/chrome:147.0.7727` 应该可以检索到适合您架构的正确镜像，但您也可以通过标签拉取特定的架构镜像。

此镜像支持的架构如下：

| 架构 | 可用 | 标签 |
| --- | --- | --- |
| x86-64 | ✅ | amd64-<版本标签> |
| arm64 | ✅ | arm64v8-<版本标签> |
| armhf | ❌ |  |
